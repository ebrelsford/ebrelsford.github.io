Introduction to GIS Software: QGIS

In-class Excercises

Part 1: Joining NTA data in QGIS

  1. Download the NTA shapefile.
  2. Download the Total Households by Household and Family Type, 2010 Excel file here.
  3. Clean the Excel file. Remove extra, unnecessary rows. With an eye toward mapping the percentage of households with married couples and chilrden under 18, delete unnecessary columns and rename column headers.
  4. In QGIS, join the cleaned Excel file with the shapefile and style the NTAs with a graduated style.

Part 2: Spatial Joins, points to polygons

  1. Download today's data here.
  2. Open the shapefile in the folder called Schools_SAT_clean. This is the schools joined with the SAT data we looked at last time. Examine the attribute table.
  3. Download the shapefile for NYC School Districts.
  4. Join by location by going to Vector > Data Management Tools > Join Attributes By Location:
    1. Make the school districts the Target vector layer.
    2. Make the schools with SAT scores the Join vector layer.
    3. Under Attribute Summary, select Take summary of intersecting features and leave Mean selected.
    4. Select a place to save the results and click OK.
  5. Examine the resulting shapefile's attribute table.
  6. Style the data using a graduated style.

Part 3: Spatial Join, polygons to points

  1. Clear the layers from the previous part.
  2. Open the shapefile in Public_School_Locations and the school districts shapefile.
  3. Join by location by going to Vector > Data Management Tools > Join Attributes By Location:
    1. This time make the schools the Target vector layer.
    2. And this time make the school districts the Join vector layer.
    3. Under Attribute Summary, select Take attributes of the first located feature.
    4. Select a place to save the results and click OK.
  4. Examine the resulting shapefile's attribute table.
  5. Style the data using a categorized style on school district number.
