Introduction to GIS Software: QGIS

In-class Excercises

Part 1: Styling with QGIS

  1. Download this data.
  2. Open the data in QGIS.
  3. Experiment with styling both polygons and points in QGIS.
  4. Create a PDF of your map using Print Composer.

Part 2: Projections

  1. Go to tagis.dep.wv.gov/convert/.
  2. Input latitude and longitude in the first input box and select an output projection (the second dropdown).
  3. Experiment with other projections, you can also click on the map to input coordinates.

Part 3: Projections in QGIS

  1. Open the AirBnB data and borough boundaries that you downloaded earlier in QGIS.
  2. Under Project > Project Properties, turn off "on the fly" CRS transformation.
  3. Zoom in and out on the map view, and the data should no longer overlap.
  4. Right click on the AirBnB layer, go to Save As..., and set the CRS (projection) to 2263 and add this data to your map view.
