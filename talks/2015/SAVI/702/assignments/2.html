A Small Scale Thematic Map

Pick a part of the world that you like or want to know more about and try your hand at making a thematic map of that area.

  1. Download Natural Earth data to make a map of the region you chose. Natural Earth is a free-to-use and accurate data set for the whole world, and it is the go-to data for this type of task. You will likely be interested in the 10m cultural and 10m physical vector files, but feel free to use any that make sense for your map. All of the files are available on the Natural Earth downloads page.
  2. Open the data in QGIS.
  3. Using the techniques described in the video above and discussed in class this week, layer the data in such a way that your theme is furthered. Example themes might be: disputed territory, Antarctic claims, and transportation infrastructure. You are free to find and incorporate other data, but it will be fine to stick with Natural Earth data for this assignment.
  4. While working on this map, use expressions in at least one way. Recall the parts of QGIS that allow you to use expressions:
    • Select by expression, to select features that you want to pull out to another layer
    • Rule-based styles, to hide features or style specific features in a certain way
    • Data defined overrides, which let you specify a style using data from a shapefile (eg to make a marker larger based on a number in the shapefile)
  5. Change your project properties to a projection that makes sense for the region you are looking at, and justify this in your submission. If you need help finding a good one, try Projection Wizard.
  6. Export your map using the QGIS print composer and submit it via email along with an example of an expression you used while working on the map.
