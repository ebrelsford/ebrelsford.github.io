Introduction to GIS Software: QGIS

In-class Excercises

Part 1: Opening CSVs in QGIS

  1. Download this data from Chicago.
  2. In QGIS, go to Layer > Add Layer > Add Delimited Text Layer, and browse for the file.
  3. Set X field and Y field to X_COORDINATE and Y_COORDINATE, respectively.
  4. Click OK, and when you are prompted for a projection go back to the page where you downloaded the data and determine which projection X_COORDINATE and Y_COORDINATE are in.
  5. Do the above again, but this time using the LATITUDE and LONGITUDE fields.

Part 2: Counting points in polygons in QGIS

  1. Download this data from the WNYC data news team. Click on the CSV in the file browser, click raw and save.
  2. Open the file in QGIS.
  3. Download and open the city council districts shapefile via Bytes of the Big Apple. Open it in QGIS.
  4. Find the count of points in by city council district: go to Vector > Analysis Tools > Points in Polygon and ensure that the city council districts are selected for the polygon layer, mean streets data for the points layer.
  5. Open the attribute table for the new shapefile. It should have a new column named PNTCNT. Is it NULL for every feature? Why might this be, and how can you fix it?
  6. Fix the problem and leave the file open for the next part.

Part 3: Styling choropleths in QGIS

  1. Using the data from the previous part, style it using a Graduated style with the new PNTCNT column.
  2. Try a few classification schemes and look at the histogram for each.

Part 4: Normalizing data in QGIS

  1. Using the data from the previous part, style it using a Graduated style with the new PNTCNT column normalized to the area of the city council districts.
  2. Again, try a few classification schemes and look at the histogram for each.
