Introduction to GIS Software: QGIS

In-class Excercises

Part 1: MapPLUTO metadata

  1. Download the MapPLUTO metadata.
  2. Download MapPLUTO data for one borough.
  3. Unzip the data.
  4. Find and open the "data dictionary" in the unzipped folder, which should have a name like PLUTODD15v1.pdf. This is a slightly more detailed version of MapPLUTO's metadata.
  5. Read through the metadata and the data dictionary to find a column you can make a categorized map out of.
  6. Open the shapefile. If the data disappears as you zoom in, save the layer as a new one and open that.
  7. Make a categorized map using the column you found.

Part 2: Selecting and Saving Data

  1. Using the map you made in Part 1, select the data for one of the categories using Select by expression.
  2. Save the selection as a new layer.

Part 3: Filtering data with expressions

  1. Start with the same data from Part 1.
  2. Style your data with a Rule-based style to accomplish the same effect as you did in Part 2.
