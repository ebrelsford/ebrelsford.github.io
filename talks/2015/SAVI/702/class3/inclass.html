Introduction to GIS Software: QGIS

In-class Excercises

Part 1: MapPLUTO metadata

  1. Download the MapPLUTO metadata.
  2. Download MapPLUTO data for one borough.
  3. Unzip the data.
  4. Find and open the "data dictionary" in the unzipped folder, which should have a name like PLUTODD15v1.pdf. This is a slightly more detailed version of MapPLUTO's metadata.
  5. Read through the metadata and the data dictionary to find a column with which you can make a categorized map.
  6. Open the shapefile. If the data disappears as you zoom in, save the layer as a new one and open that.
  7. Make a categorized map using the column you found.

Part 2: Labels with QGIS

  1. Download the 1:10m cultural vectors file from Natural Earth. This is 200+ MB, so there will be a USB drive with the data if you need it. Failing that, download just the country data.
  2. Open the country borders from Natural Earth.
  3. Add text labels with the country names.
  4. Experiment with size, color, and placement.

Part 3: Selecting and Saving Data

  1. Keep the data from Part 2 open (countries).
  2. Select data using Select by expression:
    1. Features with 'France' as the value for sovereignt.
    2. Features with 'Dependency' as the value for type.
    3. Features with name_len over 20.
    4. Another selection of your choosing.
  3. Save one of the above selections as a new layer.

Part 4: Filtering data with expressions

  1. Open the countries data from Part 2 again (non-filtered).
  2. Style the data with a Rule-based style to show only the features that you selected for one of the selections from the previous part.

Part 5: Style using expressions

  1. Open the AirBnB data we looked at last week (or download it again).
  2. Style the size of the markers based on the price attribute, dividing by a large number if the markers are too large.
