Exercise 1: styling layers

  1. Download the data for the day.
  2. Open QGIS.
  3. Open the shapefile in School Districts.
  4. Open the shapefile in Public School Locations.
  5. Style both layers.
  6. Add labels to the school districts layer.

Exercise 2: opening CSVs, making selections

  1. Open the CSV in the NYPD Motor Vehicle Collisions folder, setting the projection appropriately.
  2. Select only those features where more than one person was injured.
  3. Filter the layer to only those features where more than one person was injured.
  4. Save the filtered features to a shapefile.

Exercise 3: counting points in polygons, making choropleths

  1. Open the shapefile in School Districts.
  2. Open the filtered shapefile of NYPD Motor Vehicle Collisions from the previous exercise.
  3. Count the number of crashes in each school district: Vector > Analysis Tools > Points in polygon.
  4. Create a choropleth: style the resulting layer using a Graduated style on the count column.

Exercise 4: using print composer

  1. Open the shapefile in School Districts.
  2. Open the shapefile in Public School Locations
  3. Save the school locations to a new shapefile, changing the projection to 4326.
  4. Select one school district and save it to a new shapefile.
  5. Select all the schools in the school district from the previous step (Vector > Research Tools > Select by location) and save them to a new shapefile.
  6. Remove all layers except the selected school district and schools.
  7. Add a base map.
  8. Create a print composer, adding the map, a title, and a legend.
  9. Export a PDF of your map.
