Assignment 3

Now that you have some experience in QGIS, let's expand on that by making a few common types of maps: qualitative and quantitative maps.

Videos that reinforce the lecture and will be helpful with this assignment are posted below.

Resources

For choosing your data classification scheme for the second part, please refer to:

  • The concepts page for a Penn State class on Cartography and Visualization. This gets relevant around the heading "Classification Schemes," but the Choropleth Maps section is a pretty good review of what we talked about in class.
  • Krygier's outline for his Cartography and Visualization class. You can skip the sections dealing with ArcGIS and ArcView, the concepts are the important part. The classification schemes are listed under "Ideographic schemes" (search for it, it's around half of the way down the page).
  • Telling the Truth, a quick look at which classification schemes work better for example distributions.

Part 1: A Qualitative Thematic Map

Create a qualitative thematic map using QGIS and any of the data sources listed in class to make a qualitative map of NYC or another area interesting to you. The map should categorize the features and list those categories in its legend. The map should have little to do with the quantity of features and more to do with the information you know about these features. Think about the data you're using and why you are categorizing it the way you are.

Part 2: A Quantitative Thematic Map

Make one quantitative thematic map counting (or otherwise summarizing) points in polygons. Think about the colors you're using, the method you're using to classify the data. If there are outliers, consider the following: Do those outliers make sense to you? Are they a quirk of the data?

Select some data source where the features are points for NYC or another area (points being something similar to Uber pickups as we looked at last week, or Wifi in the Parks). Using QGIS, count the number of these points in each polygon for a polygon data source (such as city council districts or census tracts) and make a choropleth map using the number of points in each polygon. We went over this briefly in class, but the video below should suffice.

Submitting Your Work

Use the QGIS Print Composer to create two PDFs: one of your map from Part 1, the other of your map from Part 2. Make sure to add map accessories such as legends, titles, and scale bars as appropriate.

Finally, include a short summary (1/2 page) of what you did for this assignment. Explain what you created and how you did so. What did you have problems with? What did you enjoy? What would you like to know how to do next?

In both parts I want to see that you are thinking about cartographic concepts as we discussed in Class 2 and the techniques and issues that we discussed in Class 3.
