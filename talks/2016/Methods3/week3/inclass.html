In-class Exercises

Part 1a: Project Projections

  1. Download this class's data.
  2. Open the file ne_10m_admin_0_countries in QGIS.
  3. Go to Project > Project Properties.
  4. Turn on "on the fly" CRS transformation.
  5. Find and select North_America_Albers_Equal_Area_Conic.
  6. Press OK.

Part 1b: Layer Projections

  1. Open the shapefiles uber_2014_04_01 and city_council_districts in QGIS.
  2. Under Project > Project Properties, turn off "on the fly" CRS transformation.
  3. Zoom in and out on the map view, and the data should no longer overlap.
  4. Right click on the uber_2014_04_01 layer, go to Save As..., and set the CRS (projection) to 2263 and add this data to your map view.
  5. What happens now if you turn "on the fly" CRS transformation off?

Part 1c: CSV Layer Projections

  1. Open the city_council_districts in QGIS.
  2. Open the NYPD Motor Vehicle Collisions CSV in QGIS.
  3. Open the 7 Major Felony Incidents CSV in QGIS.

Part 2: Categorized maps

  1. Open the 7 Major Felony Incidents CSV in QGIS if it's not already open.
  2. Style using a Categorized style on the column Offense.

Part 3: Choropleth maps

  1. Open the shapefiles uber_2014_04_01 and city_council_districts in QGIS.
  2. Count the number of uber pickups in each council district.
  3. Style the resulting layer with a Graduated style on the column containing the counts.
