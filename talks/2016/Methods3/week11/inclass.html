In-class Exercises

Part 1: Time Manager

  1. Download this data.
  2. Open the earthquakes shapefile in QGIS.
  3. Install the TimeManager QGIS plugin.
  4. Use TimeManager to animate the earthquakes using the time column with a time frame of one day.

Part 2: OSM Metro Extracts

  1. Go to Mapzen's Metro Extracts page
  2. Select a city you would like to find data for.
  3. Download both the shapefiles: OSM2PGSQL and IMPOSM.
  4. Open the files in QGIS.

Part 3: OSMDownloader

  1. Install the OSMDownloader QGIS plugin.
  2. Zoom to a small (part of a city or smaller) area on the map, using a base map or similar to find an area you want to map.
  3. Download data by going to Vector > OpenStreetMap > Download Data. In the dialog select Extent: From map canvas. Don't forget to browse for somewhere to put the data and make sure the filename ends with .osm.
  4. Open the data—drag and drop it into OSM, select all the layers, and press OK.
