In-class Exercises

Part 1: Expressions

  1. Download today's data.
  2. Open the nynta_16c shapefile in QGIS. This is from DCP's open data page.
  3. Open the dem_2014acs5yr_nta.xlsx Excel file in QGIS by dragging and dropping it into QGIS. When asked which layer to add, select DemData. This is the Demographic data (2010-2014) by NTA Excel file from here, under profiles.
  4. Join the data by GeoID (from the Excel file) and NTACode (from the shapefile).
  5. Make a choropleth map using the joined data. Start with the percentage of the population that is below 5 years old, PopU5P.
  6. Make another choropleth map of the percentage of the population that is 14 years or younger. You will want to add the percentage columns for each age range up to and including 10 to 14 years.

Part 2: Using expressions to normalize

  1. Using the NTA demographic data as joined in Part 1, calculate the population density for each NTA and make a choropleth of it.
  2. Use the field calculator to add a population density column to the shapefile.
  3. Open the city_council_districts shapefile, save it with the projection 4326.
  4. Open the Subway Entrances shapefile.
  5. Count the number of subway entrances in each council district. Make a choropleth of it.
  6. Make a choropleth of the density (by area) of subway entrances in each council district.
