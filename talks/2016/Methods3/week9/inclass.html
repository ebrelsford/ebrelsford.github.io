In-class Exercises

Part 1: Analysis

  1. Download today's data.
  2. Upload the Graffiti Locations zip file to your CARTO account and make a map with it.
  3. Experiment with the Group points into polygons analysis.
  4. Experiment with the Filter by column value analysis.

Part 2: HTML

  1. Add a link in the legend for a map in our CARTO account.
  2. A link should look something like this: 
    <a href="https://data.cityofnewyork.us/">Data via NYC Open Data Portal</a>
  3. Edit the popup for a CARTO map. You could remove or change the label for a field or combine fields, for example.

Part 3: CSS

  1. Use CSS to style some part of your popup in a CARTO map. For example, a field might look like this: 
    <h4 class="CDB-infowindow-title">{{name}}</h4>
    Use CSS properties to change the text color: 
    <h4 class="CDB-infowindow-title" style="color: blue;">{{name}}</h4>
  2. Use font-weight to make the text not bold ("normal").
