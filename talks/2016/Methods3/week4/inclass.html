In-class Excercises

Part 1: Attribute Joins

  1. Download this class's data.
  2. Open the file Public_School_locations in QGIS.
  3. Open the CSV SAT_Results.csv in QGIS using the Add Delimited Text Layer tool.
  4. Attribute join the SAT results to the public schools.
  5. Style the schools.
  6. Save the joined layer as a new shapefile.

Part 2: Spatial Joins

  1. Keep the joined layer from Part 1 open in QGIS.
  2. Open the shapefile in city_council_districts.
  3. Use Vector > Data Management Tools > Join attributes by location to find the average SAT math score by city council district.
  4. Style the city council districts by the average score.

Part 3: Geoprocessing

  1. Clip nys_census_tracts to the borough_boundaries. That is, remove the parts of the tracts that are outside of the borough boundaries.
  2. Select resulting census tracts in a selected borough.
  3. Buffer Graffiti_Locations by 500 feet.
  4. How many Public_School_locations are within 500 feet of a point in Graffiti_Locations? Use Vector > Analysis Tools > Select by location.
