source

four steps:

  1. create a button
  2. get the map layer
  3. listen for a click on the button
  4. change the layer's SQL

look at jsbin.com/hasumo/edit

1. create a button

<button class="queens-button">
    just Queens
</button>

2. get the map layer


cartodb.createVis(...)
.done(function (vis, layers) {
    dataLayer = layers[1].getSubLayer(0);     
});

3. listen for a click on the button

$('.queens-button').click(function () {
    ...
});

4. change the layer's SQL

$('.queens-button').click(function () {
    var sql = "SELECT * FROM nypd_motor_vehicle_collisions_june_2015 WHERE borough = 'QUEENS'";
    dataLayer.setSQL(sql);
});

khan academy

variables can be numbers

var size = 5;

and you can do math on them

var size = 5 * 1000;

variables can also be strings (text)

var sql = 'SELECT * FROM collisions';

and you can add strings together

var injuries = 5;
var sql = 'SELECT * FROM collisions WHERE injuries = ' + injuries;

you can do this with any other kind of input

  1. Add the input to the page
  2. Listen for the input to change
  3. Update your data layer's SQL

let's do this with a dropdown

1. Add a dropdown

<select class="borough-picker">
    <option value="all">All boroughs</option>
    <option value="BRONX">Bronx</option>
    <option value="BROOKLYN">Brooklyn</option>
    <option value="MANHATTAN">Manhattan</option>
    <option value="QUEENS">Queens</option>
    <option value="STATEN ISLAND">Staten Island</option>
    <option value="">other</option>
</select>

2. Listen for it to change

$('.borough-picker').change(function () {
});

3. Update the SQL

$('.borough-picker').change(function () {
  var borough = $(this).val();
  var sql;
  if (borough === 'all') {
    sql = "SELECT * FROM nypd_motor_vehicle_collisions_june_2015";
  }
  else {
    sql = "SELECT * FROM nypd_motor_vehicle_collisions_june_2015 WHERE borough = '" + borough + "'";
  }
  dataLayer.setSQL(sql);
});
