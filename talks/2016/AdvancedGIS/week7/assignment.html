Part 1: JavaScript

You're not expected to fully know JavaScript for this course. However, being able to find answers to JavaScript questions and understand the fundamentals will make the next two weeks much easier for you. With that in mind, explore the introductory lessons from one of the following JavaScript courses:

  • Khan Academy
  • Codecademy
  • Eloquent JavaScript

Part 2: Use CartoDB.js on your Inside Airbnb page

Here we continue to improve our version of Inside Airbnb.

  1. Replace the embedded map on your web page you have worked on for Assignments 5, Part 3, and 6, Part 3. Use CartoDB.js to add the map in place of the iframe-embedded map. This jsbin contains an example of doing this.
  2. Make your neighborhood dropdown change which features are visible on your map. Use JavaScript to listen for changes to the dropdown and update the map's SQL accordingly. This jsbin is an example of doing this with a borough dropdown. It is okay if you only have a few of the neighborhoods in the dropdown for now.
  3. Post the update page online, preferably uing GitHub Pages.
