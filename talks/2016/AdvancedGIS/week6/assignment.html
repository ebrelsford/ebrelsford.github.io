Part 1: Add some CSS to a CartoDB visualization

Using a visualization from earlier (or a new one), use CSS to edit your infowindow and legend. Try to make both fit the style of your basemap and data.

Part 2: Improve your HTML page from Assignment 5

Using the page from Assignment 5 (Part 2), add CSS to style it. Consider fonts, colors, and positions when making your changes. Use at least three selectors, and select based on your elements' classes for at least one. Put the styled page online using Neocities or GitHub Pages.

Part 3: Improve your version of Inside Airbnb from Assignment 5

  1. Start with the HTML from Assignment 5, Part 3.
  2. Edit the HTML page:
    • Add a style element in the head of the page.
    • Style the text on the page as you see fit.
    • Style links differently than they are styled by default.
    • Style the area where you put the total number of rooms available by type ("entire home", "private room", "shared room"). Change the background and add some padding.
  3. Put the HTML page online using Neocities or GitHub Pages.

Submitting your work

Submit a link to the visualization you edited in Part 1 and links to the pages you posted online for Parts 2 and 3.

You may optionally choose to include a short paragraph response or summary of your work.
