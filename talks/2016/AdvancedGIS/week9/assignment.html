Continue working with JavaScript

There are numerous directions to go with JavaScript from this point. You are encouraged to pick one of the following activities based on which sounds most interesting to you or will be useful to you as you work on your final project. Your choices are:

(A) Add other data to your Inside Airbnb page

Here we continue to improve our version of Inside Airbnb that we've been working on for the past few weeks.

  1. Continue with your Inside Airbnb page from Assignment 7.
  2. Modify the page to show some basic statistics about the listings using the CartoDB SQL API and JavaScript. This JS Bin contains an example of finding the average price and putting it on a page. You could choose to show the range of prices (MIN and MAX) or the total number of listings or something else relevant to the map.
  3. Post the updated page online using GitHub Pages.

(B) Make a Leaflet map

Make a map outside of CartoDB but using CartoDB to get your data.

  1. Start a new HTML page.
  2. Add Leaflet to the page.
  3. Select some data from a dataset in CartoDB using an SQL query.
  4. Use that SQL query to get the data and put it on your Leaflet map. This JS Bin is a relevant example.
  5. Post the page online using GitHub Pages.

(C) Customize your CartoDB infowindows using CartoDB.js

This is particularly handy if you would like to embed content such as YouTube videos in your CartoDB infowindows, which is not possible through the CartoDB website.

  1. Start a new HTML page.
  2. Load your map on the page using CartoDB.js. See this JS Bin for an example of doing this if you need a refresher.
  3. Customize the infowindows by creating a custom template for the infowindows in your HTML, then using CartoDB.js to tell the map to use your template. This JS Bin has all the information you need to do this.
  4. Post the page online using GitHub Pages.
