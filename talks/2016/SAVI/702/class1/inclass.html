In-class Excercises

Part 1: Latitude and Longitude

  1. Let's plot some points!
    1. Visit dwtkns.com/pointplotter.
    2. Open the list of volcanoes from NOAA at the bottom of the page in a new tab.
    3. Copy and paste the coordinates for the volcanoes into the text box on dwtkns.com/pointplotter and press Plot points.
  2. Visit darrinward.com/lat-long and paste the same list.
  3. Visit bboxfinder.com.

Part 2: Install QGIS

Windows
  1. Visit the QGIS download page.
  2. Select the appropriate package (32 bit or 64 bit) and download it.
  3. Once the package has downloaded, run it.
  4. Open QGIS to confirm that it works.
OS X
  1. Visit the QGIS download page on kyngchaos.com.
  2. Download and install everything under Requirements ("GDAL Complete ..." and "Matplotlib Python module").
  3. Once those requirements are installed, download and install QGIS, under Download.
  4. Open QGIS to confirm that it works.
