In-class Excercises

Part 1: Joining NTA data

  1. Download the NTA shapefile.
  2. Download the Demographic data (2010-2014) by NTA Excel file here, under profiles.
  3. Open the shapefile in QGIS.
  4. Open the Excel file in your spreadsheet program. Look at the Ddictionary sheet for details about what the columns in the data are.
  5. Open the Excel file in QGIS by dragging and dropping it into QGIS. When asked which layer to add, select DemData.
  6. Make a choropleth map using the joined data. Start with the percentage of the population that is below 5 years old.
  7. Make another choropleth map of the percentage of the population that is 19 years or younger. You will want to add the percentage columns for each age range up to and including 15 to 19 years.

Part 2: Joining World Bank data

  1. Download Agricultural land (% of land area) from the World Bank.
  2. Download the Admin 0 — Countries (without boundary lakes) shapefile from Natural Earth.
  3. Clean the CSV file from the World Bank by deleting the first four rows (everything up to the column headers). You should not have to change anything else.
  4. In QGIS, join the cleaned CSV file with the shapefile and make a choropleth for the 2013 data.
  5. Using the Field Calculator, create a column that contains the difference in agricultural land from 2003 to 2013. Make a choropleth with the new column.
  6. Use Select by expression to select the countries in multiple regions (for example, both Europe and Asia).
